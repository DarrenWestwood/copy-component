Blockonomics Copy Component

Usage

Include CSS in Header 
<link rel="stylesheet" href="bnomics-input.css">
Create the Input Container 
<div class="bnomics-input" data-style="light|dark|dark-alt" data-icon-style="(Optional) light|dark" data-copy="(Optional) Data to Copy instead of the content inside">
    Content to Copy. Can be a HTML Content or Simple text
</div>
Include Required JS and Initialise the Container 
<script src="bnomics-input.js" type="text/javascript" />
<script>
    new BlocknomicsInput(document.querySelector(".bnomics-input"))
</script>
